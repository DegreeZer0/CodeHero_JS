### About the Game:
CodeHero_JS is a Puzzle style game with programming involved; Its main focus (for now) is in JavaScript, the first 20 phases ask only the basic movement and subsequent phases already begin to have a theme of "requirement" to be completed in the style:
* Write in the Hello World console!
* Exercise a loop ending at 5
* Write a solution if/elseif/else
* ...

The goal is to increase this game more and more to the point that it can be used as a form of teaching for those who want to learn JavaScript and other languages in a casual style of play

## How to use:
* **move** => Can be used raw to walk at angles x,y without being able to walk vertically,
* **for** => Can be set to loop execution to shorten lines of code,
* **let** => Is used to declare a variable (Can be used well in loop),
* **const** => Used to declare a fixed constant (Hint: const w = moveUp(),)
* **function** => Compiles several acts into a single defined keyword,
* **if** => Used to do motion analysis to avoid incompatibilities and crash,
* **console.log** => Purely used to show what you need in the terminal.
