## O que é o Code Hero?
O CodeHero_JS nada mais é do que um projeto desenvolvido somente por mim (DegreeZer0) enquanto estou na minha jornada FullStack de desenvolvedor web, o jogo estará em constante evolução conforme também evoluo nesta área; Pretendo trazer sempre novas mecânicas, fases, temas, desafios novos e possuo planos para mais a frente poder adicionar roadpaths de outras linguagens dentro do jogo
### Sobre o Jogo:
CodeHero_JS é um jogo estilo Puzzle com programação envolvido; Seu foco principal (Por agora) é em JavaScript, as primeiras 20 fases pedem somente o básico de movimentação e as fases subsequentes já iniciam a ter uma temática de "requirement" para ser concluída no estilo:
* Escreva no console Hello World!
* Exerça um loop com terminação em 5
* Elabore um if/elseif/else de solução
* ...

A meta é incrementar esse jogo cada vez mais a ponto que ele possa ser usado como forma de ensino para quem quer aprender JavaScript e outras linguagens em um estilo de jogo casual

## CodeHero_JS será pago?
Sendo direto: Sim... mas perai, não desanima ainda, o que será pago no CodeHero será as funções de criar custom levels, participar dos roadpaths integrados e também de poder jogar alguns custom maps que possuem requisitos que serão estabelecidos em versões futuras (Esse README.md vai ser atualizado junto).

## Guia atual:
**Funções ativas**
* moveUp(); <-- Anda pra cima
* moveDown(); <-- Anda pra baixo
* moveLeft(); <-- Anda pra esquerda
* moveRight(); <-- Anda pra direita
* for () {}; <-- Pode declarar loop
* let x = ""; <-- Declara variável
* const x = ""; <-- Declara constante
* function x() {}; <-- Insere uma função
* if (x = "") {
    return "";
  } elseif {
    return "";
  } else {
    return "";
  } <-- Avalia condições
* console.log(""); <-- Executa log no console

## Como usar:
* **move** => Pode ser usado cru para andar nos ângulos x,y sem poder andar em vertical,
* **for** => Pode ser definido para execução de loop para encurtar linhas de código,
* **let** => É usado para declarar uma variável (Pode ser usado bem em loop),
* **const** => Usado pra declarar uma constante fixa (Dica: const w = moveUp();),
* **function** => Compacta diversos atos em uma única palavra chave definida,
* **if** => Usado pra fazer análise de movimentos para evitar incompatibilidades e crash,
* **console.log** => Puramente usado pra mostrar o que precisa no terminal.
